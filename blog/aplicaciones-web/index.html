Aplicaciones web

2025-05-29

La asignatura de Aplicaciones Web tiene como objetivo introducir a los alumnos en el uso y funcionamiento de aplicaciones web, entendiendo su estructura, utilidad y manejo desde el punto de vista del usuario y del técnico.

Durante el curso se trabajan los siguientes contenidos clave:

  • Introducción a las aplicaciones web: qué son, cómo funcionan y en qué se diferencian de las aplicaciones de escritorio.

  • Navegadores y motores de búsqueda: funcionamiento, configuración y uso eficiente.

  • Introducción al diseño web básico: cosas basicas de HTML y CSS.

  • Gestores de contenido (CMS): instalación y uso básico de plataformas como WordPress y Zola.
