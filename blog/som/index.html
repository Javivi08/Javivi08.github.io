SOM

2019-11-27

La asignatura de Sistemas Operativos Monopuesto (SOM) tiene como finalidad que el alumnado aprenda a instalar, configurar y gestionar sistemas operativos en ordenadores individuales, tanto de software libre como propietario.

Durante el curso se aprenden los siguientes contenidos clave:

  • Instalación de sistemas operativos monopuesto: Windows, Linux y otros sistemas comunes.

  • Particionado y formateo de discos duros.

  • Configuración básica del sistema operativo: idioma, zona horaria, drivers, usuarios, etc...

  • Gestión de archivos y carpetas: permisos, compresión, exploración y organización del sistema de archivos.

  • Instalación y desinstalación de software.

  • Herramientas del sistema operativo: mantenimiento, copias de seguridad, restauración del sistema y gestión de recursos.

  • Uso de la terminal (en sistemas Linux): comandos básicos y scripts simples.
